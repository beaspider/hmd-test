# hmd-test

## Pre Reqs

A working Go environment

## Running the code

## Linux/Mac
`export INSTAGRAM_ACCESS_TOKEN=<OAUTH_TOKEN>`

`go get github.com/beaspider/hmd-test`

`cd $GOPATH/src/github.com/beaspider/hmd-test`

`go run main.go`

## Windows
`set INSTAGRAM_ACCESS_TOKEN=<OAUTH_TOKEN>`

`go get github.com/beaspider/hmd-test`

`cd $GOPATH%/src/github.com/beaspider/hmd-test`

`go run main.go`
